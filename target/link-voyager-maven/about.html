Una aplicación diseñada para explorar y visualizar la red de enlaces (tomando solamente los elementos HTML <a>) que conectan diferentes páginas web, comenzando desde una página específica. Permite descubrir cómo están interrelacionados los sitios a través de los hipervínculos, mostrando la estructura de navegación a partir de un punto de inicio determinado. 🤓 Además, esta app utiliza una estructura de datos basada en grafos como soporte para representar el mapa de conexiones entre páginas y enlaces. ¡La estructura fue creada por mi hermano Maxi, que es un groso total! 💻🎉
